# 🔐 Login Authentication Web Page  

A simple and modern **Login Authentication** project built using **HTML, CSS, and JavaScript**.  
This project features a clean glassmorphism design with background effects and basic client-side form validation.

---

## 📌 Features  

- ✨ Glassmorphism UI design  
- 🖼️ Full-screen background image  
- 📧 Email format validation  
- 🔑 Password length validation (minimum 8 characters)  
- 🎨 Smooth input focus and hover effects  
- 🧩 Lightweight and beginner-friendly code  

---

## 🛠️ Technologies Used  

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling and visual effects  
- **JavaScript** – Form validation logic  

---

Login-Authentication/

---

## 🚀 How It Works  

1. User enters email and password.  
2. JavaScript validates:
   - Email must contain `@` and `.` in proper format.  
   - Password must be at least 8 characters long.  
3. If validation fails, an alert message is shown.  
4. If validation passes, the form proceeds.  

---

## ▶️ How to Run  

1. Download or clone this repository.  
2. Make sure all files are in the same folder.  
3. Open `index.html` in your browser.
